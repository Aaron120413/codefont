等宽字体快速安装脚本
===============

------------

运行``./install.sh`` 安装全部字体



快速安装
------------------

```
    # clone
    git clone https://github.com/zhenruyan/codefont --depth=1
    # install
    cd fonts
    ./install.sh
    # clean-up a bit
    cd ..
    rm -rf fonts

```
